**FireantCrawler._store_posts():** Replace the per-post loop (lines 204-212) with a single bulk insert:
```python
async def _store_posts(self, ticker_id: int, posts: list[dict]) -> int:
    if not posts:
        return 0
    rows = [{"ticker_id": ticker_id, **post} for post in posts]
    stmt = insert(Rumor).values(rows).on_conflict_do_nothing(constraint="uq_rumors_post_id")
    result = await self.session.execute(stmt)
    return result.rowcount
```

**CafeFCrawler._store_articles():** Replace the per-article loop (lines 197-209) with a single bulk insert:
```python
async def _store_articles(self, ticker_id: int, articles: list[dict]) -> int:
    if not articles:
        return 0
    rows = [{"ticker_id": ticker_id, "title": a["title"], "url": a["url"], "published_at": a["published_at"]} for a in articles]
    stmt = insert(NewsArticle).values(rows).on_conflict_do_nothing(constraint="uq_news_articles_ticker_url")
    result = await self.session.execute(stmt)
    return result.rowcount
```

**F319Crawler.crawl_rss():** Accumulate all insert rows in a list during the item loop, then execute a single bulk insert after the loop:
```python
# After the for-item loop, instead of per-row inserts inside the loop:
rows_to_insert = []
for item in items:
    # ... existing ticker matching logic ...
    for symbol in matched:
        rows_to_insert.append({...rumor values...})

if rows_to_insert:
    stmt = insert(Rumor).values(rows_to_insert).on_conflict_do_nothing(constraint="uq_rumors_post_id")
    result = await self.session.execute(stmt)
    stored = result.rowcount
```

**VnExpressCrawler.crawl_rss():** Same pattern as F319 — accumulate rows, single bulk insert at end.

**VietstockCrawler.crawl_rss():** Same pattern — accumulate rows, single bulk insert at end.

Note: `rowcount` on a bulk INSERT...ON CONFLICT DO NOTHING returns the total number of actually inserted rows (not conflicted), which is exactly what we need for the `stored` count.
