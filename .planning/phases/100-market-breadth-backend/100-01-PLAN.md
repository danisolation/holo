from pydantic import BaseModel

class ADLineItem(BaseModel):
    date: str  # ISO date string

**Methods:**

`async def get_ad_line(self, start_date: date, end_date: date) -> list[dict]`:
- Query DailyPrice for ALL HOSE tickers in date range (use TickerService.get_ticker_id_map(exchange="HOSE") to get valid ticker_ids)
- Use a single SQL query with window function: for each ticker+date row, compare close to LAG(close, 1) OVER (PARTITION BY ticker_id ORDER BY date)
- Then GROUP BY date to count advancing (change > 0), declining (change < 0), unchanged (change == 0)
- Alternatively, fetch all DailyPrice rows for the ticker_ids in [start_date - 1 day buffer, end_date], load into pandas DataFrame, compute per-ticker daily change via groupby('ticker_id').close.diff(), then group by date to count advancing/declining/unchanged
- The pandas approach is simpler and consistent with existing codebase patterns (IndicatorService uses pandas)
- Return list of dicts: [{date, advancing, declining, unchanged, net}]

`async def get_ma_breadth(self, start_date: date, end_date: date) -> list[dict]`:
- JOIN DailyPrice with TechnicalIndicator on (ticker_id, date) for HOSE tickers
- For each date: count total tickers with non-NULL sma_50, count where close > sma_50, same for sma_200
- SQL approach: SELECT dp.date, COUNT(*) as total, SUM(CASE WHEN dp.close > ti.sma_50 THEN 1 ELSE 0 END) as above_ma50, ... GROUP BY dp.date
- Filter to only dates in [start_date, end_date], WHERE ti.sma_50 IS NOT NULL for the ma50 count (use separate counts to handle tickers that have sma_50 but not sma_200)
- Return list of dicts with pct_above_ma50 = round(above_ma50 / total * 100, 1)

`async def get_highs_lows(self, start_date: date, end_date: date) -> list[dict]`:
- For each date in range, determine if each ticker's close is the MAX or MIN of its trailing 252 trading days
- Fetch DailyPrice for all HOSE tickers from (start_date - 370 calendar days buffer for 252 trading days) to end_date
- Load into pandas, for each ticker compute rolling(252).max() and rolling(252).min() on close
- A new high = close equals rolling max. A new low = close equals rolling min.
- Group by date, count new_highs and new_lows, filter to [start_date, end_date]
- Return list of dicts

`async def get_all_breadth(self, start_date: date, end_date: date) -> dict`:
- Call all 3 methods, return combined dict matching MarketBreadthResponse shape

**Important implementation details:**
- All float comparisons should use float() on Decimal values before comparing
- Dates in response should be ISO format strings (date.isoformat())
- Handle edge case: if no price data exists for date range, return empty lists (not error)
- Use `from app.services.ticker_service import TickerService` to get HOSE ticker_ids

**1. Create `backend/app/api/market.py`:**

Follow the discovery.py API pattern exactly (TTLCache, async_session context manager, response_model).

```python
"""Market breadth API — market-wide health indicators for HOSE.

Returns A/D line, MA breadth (% above MA50/MA200), and 52-week highs/lows
for configurable date range. Single endpoint returns all 3 metrics.
TTLCache (300s) to avoid repeated heavy computation.
"""
from datetime import date, timedelta

from cachetools import TTLCache
from fastapi import APIRouter, Query

from app.database import async_session
from app.schemas.market_breadth import MarketBreadthResponse
from app.services.market_breadth_service import MarketBreadthService

router = APIRouter(prefix="/market", tags=["market"])

# TTLCache: 300s TTL, 16 entries for different date range combos
_breadth_cache: TTLCache = TTLCache(maxsize=16, ttl=300)


@router.get("/breadth", response_model=MarketBreadthResponse)
async def get_market_breadth(
    start_date: date | None = Query(None, description="Start date (default: 90 days ago)"),
    end_date: date | None = Query(None, description="End date (default: today)"),
) -> MarketBreadthResponse:
    """Get market breadth indicators for HOSE.

    Returns daily A/D line, MA breadth, and 52-week highs/lows.
    Default range: last 90 days.
    """
    today = date.today()
    _end = end_date or today
    _start = start_date or (_end - timedelta(days=90))

    cache_key = f"{_start}:{_end}"
    cached = _breadth_cache.get(cache_key)
    if cached is not None:
        return cached

    async with async_session() as session:
        service = MarketBreadthService(session)
        result = await service.get_all_breadth(_start, _end)

    response = MarketBreadthResponse(**result)
    _breadth_cache[cache_key] = response
    return response
```
