```python
async def weekly_behavior_analysis():
    logger.info("=== WEEKLY BEHAVIOR ANALYSIS START ===")
    async with async_session() as session:
        job_svc = JobExecutionService(session)
        execution = await job_svc.start("weekly_behavior_analysis")
        try:
            service = BehaviorService(session)
            result = await service.run_weekly_analysis()
            await job_svc.complete(execution, status="success", result_summary=result)
            await session.commit()
        except Exception as e:
            if execution.status == "running":
                await job_svc.fail(execution, error=str(e))
                await session.commit()
            raise
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → API | Untrusted input: target_pnl, prompt response, page/page_size params |
| scheduler → DB | Automated job writes — must handle concurrent access (prompt expiration race) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-47-01 | Tampering | POST /api/goals | mitigate | Pydantic GoalCreate schema with Field(gt=0, le=1B) — validated before handler runs |
| T-47-02 | Tampering | POST /api/goals/weekly-prompt/{id}/respond | mitigate | Pydantic Literal["cautious","unchanged","aggressive"] — rejects any other value |
| T-47-03 | Tampering | GET /api/goals/history | mitigate | No user-controlled sort params — always sorted by month DESC server-side |
| T-47-06 | Repudiation | Scheduler job failures | mitigate | JobExecutionService records start/complete/fail for every job run — full audit trail |
</threat_model>
