       - Guard: if target_pnl <= 0, return {"percentage": 0.0, "color": "red"}
       - percentage = (actual_pnl / target_pnl) * 100, capped at 200.0

       - color = "green" if pct >= 100 else "amber" if pct >= 50 else "red"

    5. `async def set_goal(self, target_pnl: Decimal) -> TradingGoal`:
       - Calculate current month start: date.today().replace(day=1)

       - If no, create new TradingGoal(target_pnl=target_pnl, month=month_start, status="active")
       - Flush and return the goal

    6. `async def get_current_goal(self) -> dict | None`:
       - Get current month start (date.today().replace(day=1))
       - Query active TradingGoal where month == current_month_start
       - If not found, return None
       - Compute actual_pnl via _compute_actual_pnl(month_start, next_month_start)
       - Compute progress via compute_goal_progress(actual_pnl, target_pnl)
       - Return dict matching GoalResponse shape

    7. `async def get_goal_history(self, page: int = 1, page_size: int = 10) -> dict`:
       - Query all TradingGoal rows ordered by month DESC, paginated
       - For each, compute actual_pnl and progress
       - Return {"goals": [...], "total": count}

    8. `async def _compute_actual_pnl(self, month_start: date, month_end_exclusive: date) -> Decimal`:
       - SELECT COALESCE(SUM(Trade.net_pnl), 0) FROM trade_journal WHERE side='SELL' AND trade_date >= month_start AND trade_date < month_end_exclusive
       - Return scalar result

    9. `async def get_pending_prompt(self) -> WeeklyPrompt | None`:
       - Query WeeklyPrompt where response IS NULL ORDER BY created_at DESC LIMIT 1
       - Return the prompt or None

    10. `async def create_weekly_prompt(self) -> WeeklyPrompt`:
        - Expire any existing pending prompts: UPDATE weekly_prompts SET response='expired' WHERE response IS NULL
        - Get current risk_level from UserRiskProfile (first row)
        - Calculate week_start: today - timedelta(days=today.weekday()) (Monday)
        - Create new WeeklyPrompt(week_start=week_start, risk_level_before=risk_level)
        - Flush and return

        - new_level = clamp_risk_level(prompt.risk_level_before, delta)
        - Update prompt: response=response, risk_level_after=new_level
        - Update UserRiskProfile.risk_level = new_level
        - Flush and return updated prompt

    12. `async def get_latest_review(self) -> WeeklyReview | None`:
        - Query WeeklyReview ORDER BY week_end DESC LIMIT 1
        - Return the review or None

    13. `async def get_review_history(self, page: int = 1, page_size: int = 10) -> dict`:
        - Query all WeeklyReview rows ordered by week_end DESC, paginated
