```python
"""Technical indicator computation and storage service.

Computes RSI(14), MACD(12,26,9), SMA(20/50/200), EMA(12/26), BB(20,2)
using the ta library. Stores results in technical_indicators table.

Key decisions (from CONTEXT.md):
- Compute for most recent 60 days per run (settings.indicator_compute_days)
- Only compute new/missing dates (query MAX(date) per ticker)
- NaN warm-up values → NULL in PostgreSQL (fillna=False)
- SMA(200) needs 200+ rows; skip tickers with < 20 rows
"""
```

**Class structure:**

```python
class IndicatorService:
    def __init__(self, session: AsyncSession):
        self.session = session

    async def compute_all_tickers(self) -> dict:
        """Compute indicators for all active tickers.
        Returns: {success, failed, skipped, failed_symbols}
        """

    async def compute_for_ticker(self, ticker_id: int, symbol: str) -> int:
        """Compute indicators for a single ticker. Returns rows stored."""

    def _compute_indicators(self, close: pd.Series) -> dict[str, pd.Series]:
        """Pure computation — all 12 indicators from close prices."""

    async def _get_last_computed_date(self, ticker_id: int) -> date | None:
        """Get last computed indicator date for a ticker."""

    @staticmethod
    def _safe_decimal(value) -> Decimal | None:
        """Convert float to Decimal, NaN→None. Use round(value, 6) to avoid float precision issues."""
```

**Implementation details:**

1. **`compute_all_tickers()`**: Get ticker map via `TickerService.get_ticker_id_map()`. Loop through all tickers, call `compute_for_ticker()` for each. Track success/failed/skipped counts. NO delay needed (all local computation, no external API). Commit after all tickers.

2. **`compute_for_ticker(ticker_id, symbol)`**:
   - Query `daily_prices` for this ticker, ordered by date ASC: `SELECT date, close WHERE ticker_id = :id ORDER BY date`
   - If < 20 rows: log warning, return 0 (not enough for any meaningful indicator)
   - If < 200 rows: log info (SMA(200) will be NULL but other indicators compute fine)
   - Create DataFrame with `date` and `close` columns
   - Convert close to float: `df["close"].astype(float)` (ta library needs float, not Decimal)
   - Call `_compute_indicators(close_series)`
   - Get `_get_last_computed_date(ticker_id)` to find cutoff
   - Only store rows where `date > last_computed_date` (or all if None)
   - Limit to most recent `settings.indicator_compute_days` (60) dates
   - Use INSERT ... ON CONFLICT (uq_technical_indicators_ticker_date) DO UPDATE for each row
   - Return count of rows stored

3. **`_compute_indicators(close)`** — instantiate each ta class ONCE (optimization from RESEARCH.md):

```python
"""AI-powered analysis service using Google Gemini.

Sends batched ticker data to Gemini for technical and fundamental scoring.
Uses structured output (response_schema) for type-safe responses.

Key decisions (from CONTEXT.md):
- gemini-2.0-flash model, 15 RPM free tier
- Batch 10 tickers per prompt, 4-second delay between calls
- Retry 3x with exponential backoff (tenacity)
- Failed batches logged and skipped — partial analysis > none
- Technical input: 5-day indicator values, price vs MAs, MACD crossover state, RSI zone
- Fundamental input: P/E, P/B, ROE, ROA, revenue/profit growth, D/E, current ratio
"""
```

**Class structure:**

```python
class AIAnalysisService:
    def __init__(self, session: AsyncSession, api_key: str | None = None):
        self.session = session
        key = api_key or settings.gemini_api_key
        if not key:
            raise ValueError("GEMINI_API_KEY is required. Set in .env or pass api_key parameter.")
        self.client = genai.Client(api_key=key)
        self.model = settings.gemini_model
        self.batch_size = settings.gemini_batch_size
        self.delay = settings.gemini_delay_seconds

    async def analyze_all_tickers(self, analysis_type: str = "both") -> dict:
        """Run AI analysis for all active tickers.
        analysis_type: 'technical', 'fundamental', or 'both'
        Returns: {technical: {success, failed}, fundamental: {success, failed}}
        """

    async def run_technical_analysis(self) -> dict:
        """Technical analysis for all tickers. Returns {success, failed, failed_symbols}."""

    async def run_fundamental_analysis(self) -> dict:
        """Fundamental analysis for all tickers. Returns {success, failed, failed_symbols}."""

    async def _analyze_technical_batch(self, ticker_data: dict[str, dict]) -> TechnicalBatchResponse | None:
        """Analyze a batch of tickers for technical signals."""

    async def _analyze_fundamental_batch(self, ticker_data: dict[str, dict]) -> FundamentalBatchResponse | None:
        """Analyze a batch of tickers for fundamental health."""

    @retry(...)
    async def _call_gemini(self, prompt: str, response_schema) -> response:
        """Call Gemini with retry on rate limit / server errors."""

    async def _get_technical_context(self, ticker_id: int, symbol: str) -> dict | None:
        """Get 5-day indicator values for a ticker."""

    async def _get_fundamental_context(self, ticker_id: int, symbol: str) -> dict | None:
        """Get latest financial ratios for a ticker."""

    def _build_technical_prompt(self, ticker_data: dict[str, dict]) -> str:
        """Build prompt for technical analysis batch."""

    def _build_fundamental_prompt(self, ticker_data: dict[str, dict]) -> str:
        """Build prompt for fundamental analysis batch."""

    async def _store_analysis(self, ticker_id: int, analysis_type: AnalysisType,
                              analysis_date: date, signal: str, score: int,
                              reasoning: str, raw_response: dict | None) -> None:
        """Store analysis result with upsert."""
```

**Implementation details:**

1. **`_call_gemini(prompt, response_schema)`** — decorated with tenacity:
   ```python
   @retry(
       stop=stop_after_attempt(settings.gemini_max_retries),
       wait=wait_exponential(multiplier=2, min=4, max=30),
       retry=retry_if_exception_type((ClientError, ServerError)),
       reraise=True,
   )
   async def _call_gemini(self, prompt: str, response_schema):
       response = await self.client.aio.models.generate_content(
           model=self.model,
           contents=prompt,
           config=types.GenerateContentConfig(
               response_mime_type="application/json",
               response_schema=response_schema,
               temperature=0.2,  # Low for consistent analysis
               max_output_tokens=4096,
           ),
       )
       return response
   ```
   **CRITICAL:** Use `client.aio.models` (async), NOT `client.models` (sync). Sync blocks FastAPI's event loop.
   **Import:** `from google.genai.errors import ClientError, ServerError`

2. **`run_technical_analysis()`**:
   - Get ticker map via TickerService
   - For each ticker, call `_get_technical_context()` to get 5-day indicator window
   - Batch tickers into groups of `self.batch_size` (10)
   - For each batch: call `_analyze_technical_batch()`, then `asyncio.sleep(self.delay)`
   - Store each result via `_store_analysis()`
   - Skip batches that fail after retry (log error, continue)
   - Return {success, failed, failed_symbols}

3. **`_get_technical_context(ticker_id, symbol)`** — query last 5 days of technical_indicators:
   ```python
   result = await self.session.execute(
       select(TechnicalIndicator)
       .where(TechnicalIndicator.ticker_id == ticker_id)
       .order_by(TechnicalIndicator.date.desc())
       .limit(5)
   )
   rows = result.scalars().all()
   ```
   If no rows, return None (skip this ticker). Build dict with:
   - `rsi_14`: list of last 5 values
   - `macd_line`, `macd_signal`, `macd_histogram`: last 5 values
   - `sma_20`, `sma_50`, `sma_200`: latest value + `price_vs_sma20` ratio
   - `bb_upper`, `bb_middle`, `bb_lower`: latest values
   - `rsi_zone`: "oversold" (<30), "overbought" (>70), or "neutral"
   - `macd_crossover`: "bullish" if histogram > 0 and was <= 0, "bearish" if opposite, else "none"

4. **`_get_fundamental_context(ticker_id, symbol)`** — query latest financial from financials table:
   ```python
   result = await self.session.execute(
       select(Financial)
       .where(Financial.ticker_id == ticker_id)
       .order_by(Financial.year.desc(), Financial.quarter.desc().nulls_last())
       .limit(1)
   )
   row = result.scalar_one_or_none()
   ```
   Return dict with pe, pb, roe, roa, revenue_growth, profit_growth, current_ratio, debt_to_equity (all as float or None).

5. **`_build_technical_prompt(ticker_data)`** — produce a prompt like:

   ```
   You are a Vietnamese stock market technical analyst. Analyze the following tickers based on their technical indicators from the last 5 trading days.

   For each ticker, provide:
   - signal: one of strong_buy, buy, neutral, sell, strong_sell
   ```

   ```
   For each ticker, provide:
   ```

7. **`_store_analysis()`** — upsert to ai_analyses:
   ```python
   stmt = insert(AIAnalysis).values(
       ticker_id=ticker_id,
       analysis_type=analysis_type,
       analysis_date=analysis_date,
       signal=signal,
       score=score,
   ```
