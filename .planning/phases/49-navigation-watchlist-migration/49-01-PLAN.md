1. Create `backend/app/schemas/watchlist.py`:
   ```python
   """Pydantic schemas for watchlist API endpoints."""
   from pydantic import BaseModel, Field

   class WatchlistItemResponse(BaseModel):
       """Single watchlist item with optional AI signal enrichment."""
       symbol: str
       created_at: str
       ai_signal: str | None = None      # e.g. "buy", "sell", "hold"
       ai_score: int | None = None        # 1-10
       signal_date: str | None = None     # ISO date of the signal

   class WatchlistAddRequest(BaseModel):
       """POST /api/watchlist request body."""
       symbol: str = Field(min_length=1, max_length=10, description="Ticker symbol to add")

   class WatchlistMigrateRequest(BaseModel):
       """POST /api/watchlist/migrate — bulk add from localStorage migration."""
       symbols: list[str] = Field(max_length=50, description="List of ticker symbols to migrate")
   ```

2. Create `backend/app/api/watchlist.py`:
   - `router = APIRouter(prefix="/watchlist", tags=["watchlist"])`
   - **GET `/`** (response_model=list[WatchlistItemResponse]):
     - Query all UserWatchlist rows ordered by created_at desc
     - For each symbol, LEFT JOIN with ai_analyses to get the latest "combined" analysis (subquery: max analysis_date per ticker where analysis_type='combined')
     - The JOIN path: UserWatchlist.symbol → Ticker.symbol → Ticker.id → AIAnalysis.ticker_id
     - Return list with ai_signal, ai_score, signal_date populated from the join
     - Use `async with async_session() as session:` pattern (matching existing codebase)
   - **POST `/`** (response_model=WatchlistItemResponse):
     - Accept WatchlistAddRequest body
     - Uppercase the symbol: `symbol = body.symbol.upper().strip()`
     - Check if symbol already exists in watchlist (return existing if so — idempotent)
     - Insert new UserWatchlist row
     - Return the created item (without AI signal — client can refetch list)
   - **DELETE `/{symbol}`** (status_code=204):
     - Find and delete UserWatchlist where symbol matches (uppercased)
     - If not found, return 404
   - **POST `/migrate`** (response_model=list[WatchlistItemResponse]):
     - Accept WatchlistMigrateRequest body
     - For each symbol in symbols list: uppercase, strip, skip if already exists
     - Bulk insert new entries (use individual inserts in a loop — simpler, <50 items)
     - Return full enriched watchlist (same as GET /)
     - This endpoint is for one-time localStorage→DB migration

3. Register in `backend/app/api/router.py`:
   - Add import: `from app.api.watchlist import router as watchlist_router`
   - Add: `api_router.include_router(watchlist_router)`
