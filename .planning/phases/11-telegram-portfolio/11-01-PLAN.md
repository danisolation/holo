**1. Add `get_ticker_pnl(symbol)` method to `PortfolioService`** (after `get_trades`, before private helpers section):

```python
async def get_ticker_pnl(self, symbol: str) -> dict:
    """Get detailed P&L for a single ticker with FIFO lot breakdown."""
```

Implementation:
- Call `self._resolve_ticker(symbol)` — raises ValueError if not found
- Query `Lot` where `ticker_id=X AND remaining_quantity > 0`, ordered by `buy_date ASC, id ASC`
- Get latest close price from `DailyPrice` (same pattern as `get_holdings`)
- For each open lot, compute: lot unrealized P&L = `(market_price - buy_price) * remaining_quantity`
- Query realized P&L: `SELECT COALESCE(SUM(price * quantity), 0) FROM trades WHERE ticker_id=X AND side='SELL'` minus cost of consumed lots `SUM(buy_price * (quantity - remaining_quantity))` from Lot table minus sell fees
- Return dict:
```python
{
    "symbol": ticker.symbol,
    "name": ticker.name,
    "quantity": total_remaining_qty,  # sum of remaining_quantity across open lots
    "avg_cost": weighted_avg_cost,    # total_cost / total_qty
    "market_price": float or None,
    "unrealized_pnl": float or None,
    "unrealized_pnl_pct": float or None,
    "realized_pnl": float,
    "lots": [
        {
            "buy_date": lot.buy_date.isoformat(),
            "buy_price": float(lot.buy_price),
            "quantity": lot.quantity,           # original qty
            "remaining": lot.remaining_quantity,
            "lot_pnl": float or None,          # (market - buy_price) * remaining
            "lot_pnl_pct": float or None,
        },
        ...
    ]
}
```

```python
async def buy_command(update: Update, context: ContextTypes.DEFAULT_TYPE) -> None:
    """/buy <ticker> <qty> <price> [fee] — Record a buy trade."""
```
- Validate: `len(context.args) >= 3` (3 required + 1 optional fee)
- Parse: `symbol = args[0].upper().strip()`, `quantity = int(args[1])`, `price = float(args[2].replace(",", ""))`, `fees = float(args[3].replace(",", "")) if len > 3 else 0`
- On parse error (ValueError/IndexError): reply with `usage_error("/buy", "/buy VNM 100 85000")`
- `async with async_session() as session:` → `PortfolioService(session).record_trade(symbol=symbol, side="BUY", quantity=quantity, price=price, trade_date=date.today(), fees=fees)`
- On `ValueError` from service: reply with `ticker_not_found(symbol)` (covers both bad ticker and any validation error)
- On success: reply with `MessageFormatter.trade_recorded(result)`
- **CRITICAL:** Do NOT call `session.commit()` — `record_trade()` commits internally
