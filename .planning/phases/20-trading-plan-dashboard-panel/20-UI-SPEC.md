### Recommended Direction Highlight

The column matching `recommended_direction` receives:
- Left border: `border-l-2 border-[#26a69a]` (if LONG) or `border-l-2 border-[#ef5350]` (if BEARISH)
- Background tint: `bg-[#26a69a]/5` (if LONG) or `bg-[#ef5350]/5` (if BEARISH)
- "Khuyến nghị" badge: `Badge variant="secondary"` with direction-appropriate color, placed next to the direction subheading

The non-recommended column uses default card background with no border highlight.

Source: Agent's discretion — matches existing `support-resistance-card.tsx` responsive pattern. Tabs rejected in favor of stacking to maintain information density without hidden content.

## Interaction States

### Loading State

Full-height skeleton matching panel dimensions:
```tsx
<Skeleton className="h-[320px] rounded-xl" />
```
Source: Follows existing pattern — `<Skeleton className="h-[220px] rounded-xl" />` in S/R card, scaled up for taller panel.

### Empty State (no trading signal data)

Displayed when `fetchTradingSignal()` returns null/404 or when no trading_signal data exists:

```
┌─────────────────────────────────────────────┐
│                                             │
│     Chưa có kế hoạch giao dịch             │  ← text-sm text-muted-foreground
│     Dữ liệu sẽ có sau khi phân tích AI     │  ← text-xs text-muted-foreground/60
│     chạy xong.                              │
│                                             │
└─────────────────────────────────────────────┘
```

Component structure:
```tsx
<Card>
  <CardContent className="py-8 text-center">
    <p className="text-sm text-muted-foreground">Chưa có kế hoạch giao dịch</p>
    <p className="text-xs text-muted-foreground/60 mt-1">Dữ liệu sẽ có sau khi phân tích AI chạy xong.</p>
  </CardContent>
</Card>
```

Source: CONTEXT.md locked copy "Chưa có kế hoạch giao dịch" + existing empty state pattern from `support-resistance-card.tsx` line 30-41.

### Invalid Signal State (confidence score = 0)

When a direction has `confidence: 0` (validation failure), that column shows:

```
┌──────────────────────────┐
│  LONG                    │
│                          │
│  Tín hiệu không hợp lệ  │  ← text-sm text-muted-foreground, centered
│                          │
└──────────────────────────┘
```

Entire column rendered with `opacity-60` and muted styling. No price rows displayed.

Source: CONTEXT.md locked decision.

### Error State (API failure)

Not rendered as a separate state — if the fetch fails, the panel simply does not appear (consistent with existing `analysisSummary?.combined` conditional rendering pattern on the ticker page). No error card needed since the trading signal is supplementary data.

### No Animation/Transitions

No explicit enter/exit animations. Rely on default browser rendering. Keeps consistency with existing cards on the page which have no animation.

Source: Agent's discretion — minimal approach matching existing page behavior.

## Price & Number Formatting

| Data | Format | Example | Implementation |
