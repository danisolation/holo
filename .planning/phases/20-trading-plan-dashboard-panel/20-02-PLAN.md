Create `frontend/src/components/trading-plan-panel.tsx` — a new file with a single exported `TradingPlanPanel` component.

**Component structure:**

```tsx
"use client";

import { Card, CardHeader, CardTitle, CardContent } from "@/components/ui/card";
import { Badge } from "@/components/ui/badge";
import { ScoreBar } from "@/components/analysis-card";
import type { TickerTradingSignal, DirectionAnalysis } from "@/lib/api";
```

**Props interface:**
```typescript
interface TradingPlanPanelProps {
  data: TickerTradingSignal;
}
```

**Main component (`export function TradingPlanPanel`):**

Renders a `Card` with:
- `CardHeader` containing `CardTitle` with text "Kế Hoạch Giao Dịch" using `className="text-lg font-semibold"`
- `CardContent` containing a `div` with `className="grid grid-cols-1 md:grid-cols-2 gap-4"` holding two `DirectionColumn` sub-components:
  - Left: `data.long_analysis` with `isRecommended={data.recommended_direction === "long"}`
  - Right: `data.bearish_analysis` with `isRecommended={data.recommended_direction === "bearish"}`

**DirectionColumn sub-component (internal, not exported):**

Props: `{ analysis: DirectionAnalysis; isRecommended: boolean }`

Vietnamese label formatting helper (local to file):
```typescript
const fmt = (v: number) => Math.round(v).toLocaleString("vi-VN");
```

Timeframe display mapping:
```typescript
const TIMEFRAME_LABELS: Record<string, string> = {
  swing: "Swing (3-15 ngày)",
  position: "Position (vài tuần+)",
};
```

**DirectionColumn rendering logic:**

1. **Container div** with dynamic classes:
   - Base: `"rounded-lg p-3"`
   - If recommended AND direction is "long": add `"border-l-2 border-[#26a69a] bg-[#26a69a]/5"`
   - If recommended AND direction is "bearish": add `"border-l-2 border-[#ef5350] bg-[#ef5350]/5"`
   - If NOT recommended: no border/background highlight

2. **Column header row** — `div className="flex items-center gap-2 mb-2"`:
   - Direction label: `p className="text-[10px] font-semibold uppercase tracking-wider text-muted-foreground"`
     - If direction is "long": text "LONG"
     - If direction is "bearish": text "XU HƯỚNG GIẢM"
   - If `isRecommended`: render a `Badge variant="secondary"` with text "Khuyến nghị"
     - Badge className for LONG: `"text-[#26a69a] bg-[#26a69a]/10 text-[10px]"`
     - Badge className for BEARISH: `"text-[#ef5350] bg-[#ef5350]/10 text-[10px]"`

3. **Confidence section** — `div className="mb-3"`:
   - Label: `p className="text-xs text-muted-foreground mb-1"` with text "Độ tin cậy"
   - `<ScoreBar score={analysis.confidence} />`

4. **Invalid signal check**: If `analysis.confidence === 0` (which shouldn't happen per backend schema `ge=1`, but defensive), render:
   ```tsx
   <p className="text-sm text-muted-foreground text-center py-4 opacity-60">
     Tín hiệu không hợp lệ
   </p>
   ```
   And skip rendering the price rows + reasoning below.

5. **Price rows** — Series of dense rows following `support-resistance-card.tsx` pattern:
   Each row is `div className="flex items-center justify-between py-1.5"`:
   - Label: `span className="text-xs font-medium text-muted-foreground"`
   - Value: `span className="font-mono text-sm font-semibold"`

   Rows in order:
   | Label | Value | Notes |
   |-------|-------|-------|
   | "Giá vào" | `fmt(plan.entry_price)` | |
   | "Cắt lỗ" | `fmt(plan.stop_loss)` | Add `text-[#ef5350]` to value span |
   | "Chốt lời 1" | `fmt(plan.take_profit_1)` | Add `text-[#26a69a]` to value span |
   | "Chốt lời 2" | `fmt(plan.take_profit_2)` | Only render if `plan.take_profit_2` differs from `plan.take_profit_1` by more than 1; add `text-[#26a69a]` to value span |
   | "Tỷ lệ R:R" | `` `1:${plan.risk_reward_ratio.toFixed(1)}` `` | |
   | "Khối lượng" | `` `${Math.round(plan.position_size_pct)}%` `` | |
   | "Khung thời gian" | `TIMEFRAME_LABELS[plan.timeframe] ?? plan.timeframe` | |

   Where `plan = analysis.trading_plan`.

6. **Reasoning section** — `div className="mt-3 pt-3 border-t border-border"`:
   - `p className="text-xs leading-relaxed text-muted-foreground"` containing `analysis.reasoning`
