## Typography

| Role | Size | Weight | Line Height | Tailwind Class |
|------|------|--------|-------------|----------------|
| Body | 14px | 400 (regular) | 1.6 | `text-sm leading-relaxed` |
| Label | 12px | 500 (medium) | 1.4 | `text-xs font-medium` |
| Heading | 24px | 700 (bold) | 1.25 | `text-2xl font-bold tracking-tight` |
| Mono | 14px | 600 (semibold) | 1.4 | `font-mono text-sm font-semibold` |

Notes:
- Body text uses `text-sm` (14px) consistently across existing components (analysis-card, trading-plan-panel)
- Mono weight for all price/number values, matching existing `font-mono text-sm font-semibold` pattern in trading-plan-panel.tsx
- Labels use `text-xs font-medium text-muted-foreground` matching existing pattern
- Heading matches home page `text-2xl font-bold tracking-tight`
- Vietnamese explanation text: `text-sm leading-relaxed text-muted-foreground` (200-300 words needs generous line height)

## Interaction States

### Loading State (Pick Cards)

| Element | Skeleton Spec |
|---------|---------------|
| Pick card grid | 3 skeleton cards matching pick card dimensions |
| Each skeleton card | `Skeleton className="h-[400px] rounded-xl"` (approximate height of full pick card) |
| Page header | Render immediately (not dependent on API) |

### Empty State (No Picks Today)

Displayed when `GET /api/picks/today` returns 0 picks.

| Element | Spec |
|---------|------|
| Container | `Card` centered within grid area |
| Icon | `Calendar` (lucide) `size-12 text-muted-foreground/40` |
| Heading | "Chưa có gợi ý hôm nay" — `text-lg font-semibold` |
| Body | "Gợi ý sẽ được tạo sau khi phân tích AI chạy xong (~17:00 mỗi ngày giao dịch)." — `text-sm text-muted-foreground mt-2 max-w-md text-center` |
| Action | None (automated process, no user action needed) |

### Error State (API Failure)

Displayed when `GET /api/picks/today` returns error.

| Element | Spec |
|---------|------|
| Container | `Card` centered within grid area |
| Icon | `AlertTriangle` (lucide) `size-12 text-destructive/60` |
| Heading | "Không thể tải gợi ý" — `text-lg font-semibold` |
| Body | "Đã xảy ra lỗi khi tải dữ liệu. Vui lòng thử lại." — `text-sm text-muted-foreground mt-2` |
| Action | `Button variant="outline"` label "Thử lại" — calls `refetch()` |

### Live Price States

| State | Display |
|-------|---------|
| Connected + price available | `font-mono text-sm font-semibold` + colored P&L badge |
| Connected + positive P&L | Price in default color + `Badge` with `text-[#26a69a] bg-[#26a69a]/10`: "▲ +2.3%" |
| Connected + negative P&L | Price in default color + `Badge` with `text-[#ef5350] bg-[#ef5350]/10`: "▼ -1.2%" |
| Connected + zero P&L | Price in default color + `Badge variant="outline"`: "— 0.0%" |
| Market closed / no data | `text-muted-foreground`: "—" (em dash, no badge) |

### Profile Settings States

| State | Display |
|-------|---------|
| Loading profile | `Skeleton` in dialog body |
| Save in progress | Button shows spinner, disabled |
| Save success | Dialog closes, picks refetch (React Query invalidation) |
| Save error | Inline error text below save button: "Không thể lưu. Vui lòng thử lại." `text-sm text-destructive` |

## Number Formatting

| Type | Format | Example | Implementation |
|------|--------|---------|----------------|
| VND price | Thousands separator, no decimals | "60,000" | `formatVND()` from `lib/format.ts` |
| VND total | Thousands separator, no decimals | "12,000,000" | `formatVND()` from `lib/format.ts` |
| Percentage | 1 decimal place + "%" suffix | "24.0%" | `toFixed(1) + "%"` |
| R:R ratio | "1:" prefix + 1 decimal | "1:2.8" | `"1:" + ratio.toFixed(1)` |
| Composite score | 1 decimal / 100 | "73.2" | `score.toFixed(1)` |
| Share count | Integer, no separator | "200" | `Math.floor()` |
| P&L percentage | "▲"/"▼" prefix + sign + 1 decimal + "%" | "▲ +2.3%" / "▼ -1.2%" | Custom format |

## Accessibility

| Element | Requirement |
|---------|-------------|
| Pick cards | Each card is a semantic `article` or `section` with `aria-label="{symbol} — gợi ý #{rank}"` |
| Live price P&L | `aria-live="polite"` on P&L badge for screen reader updates |
| Almost-selected accordion | Native Accordion from base-ui provides keyboard + ARIA automatically |
| Profile dialog | Native Dialog from base-ui provides focus trap + Escape close |
| Color-coded prices | Never rely on color alone — always include ▲/▼ prefix for P&L direction |
| Settings gear icon | `sr-only` text: "Cài đặt hồ sơ" |
| Score bar | Existing `ScoreBar` already renders numeric score alongside visual bar |

## Data Flow

| Hook | API Endpoint | staleTime | Usage |
|------|-------------|-----------|-------|
| `useDailyPicks()` | `GET /api/picks/today` | 5 min | Pick cards + almost-selected list |
| `usePickHistory()` | `GET /api/picks/history` | 5 min | Not used this phase (Phase 45) |
| `useProfile()` | `GET /api/profile` | 10 min | Profile settings card, page subtitle |
| `useUpdateProfile()` | `PUT /api/profile` | mutation | Save profile, invalidates `["profile"]` + `["picks", "today"]` |
| `useRealtimePrices()` | WebSocket | real-time | Live price on pick cards |
