## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, score bar height (1.5 = 6px rounded to nearest) |
| sm | 8px | Compact element spacing, table cell internal padding |
| md | 16px | Default element spacing, filter gap, card padding |
| lg | 24px | Section padding, gap between filter bar and table |
| xl | 32px | Page heading to content gap |
| 2xl | 48px | Not used this phase |
| 3xl | 64px | Not used this phase |

Exceptions: Score bar track height is 6px (`h-1.5`) — established pattern from existing `ScoreCell` component.

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px (text-sm) | 400 (normal) | 1.5 |
| Label | 12px (text-xs) | 500 (medium) | 1.5 |
| Heading | 24px (text-2xl) | 700 (bold) | 1.2 |

Source: Matches existing watchlist page pattern — `text-2xl font-bold` heading, `text-sm` table body, `text-xs font-mono` for numeric scores.

### Score Colors (semantic, from existing analysis-card.tsx)

| Condition | Color | Usage |
|-----------|-------|-------|
| Score ≥ 7 (strong signal) | `#26a69a` (teal/green) | Score bar fill, signal badge |
| Score 4–6.9 (moderate) | `amber-500` | Score bar fill |
| Score < 4 (weak) | `#ef5350` (red) | Score bar fill |

Accent reserved for: "Thêm" (add to watchlist) button only. "Đã thêm" (already added) uses `variant="secondary"`.

## Score Bar Specification

```
┌──────────────────────────┐
│ ████████░░░░░░░░░░  7.5  │  ← green (#26a69a), 75% width
│ █████░░░░░░░░░░░░░  5.0  │  ← amber (amber-500), 50% width
│ ██░░░░░░░░░░░░░░░░  2.0  │  ← red (#ef5350), 20% width
│ —                        │  ← null value (text-muted-foreground dash)
└──────────────────────────┘
```

- Track: `h-1.5 bg-muted rounded-full` (6px height)
- Fill: `h-full rounded-full` with color based on value threshold
- Value: `font-mono text-xs w-6 text-right` (right-aligned numeric)
- Null: em-dash `—` in `text-xs text-muted-foreground`

## Data Flow

```
useDiscovery({ sector, signal_type })
  → GET /api/discovery?sector=X&signal_type=Y
  → DiscoveryItem[]
  → DiscoveryTable renders rows

useWatchlist()
  → Set<string> of watchlist symbols
  → per-row button state: "Thêm" vs "Đã thêm"

useAddToWatchlist()
  → POST /api/watchlist { symbol }
  → invalidates ["watchlist"] query
  → button auto-updates via useMemo Set recalculation
```

Query config: `staleTime: 5 * 60 * 1000` (5 minutes — same as market overview).
