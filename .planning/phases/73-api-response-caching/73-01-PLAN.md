Actually, for maximum cache benefit, check cache BEFORE the ticker lookup using `symbol.upper()` as key. If the symbol is invalid, it won't be in cache so we'll fall through to the DB path which will 404 naturally. This avoids even the ticker lookup on cache hit.

```python
cache_key = symbol.upper()
cached = _summary_cache.get(cache_key)
if cached is not None:
    return cached
```
