Implement automatic position management for the simulator: auto-sell when daily close price hits Stop Loss or Take Profit targets, and auto-sell when AI generates a bearish signal for open positions.

Purpose: Positions currently require manual selling. This automates exit logic based on the trading plan's SL/TP targets and AI sell signals, making the simulator realistic.
Output: Backend service methods + scheduler job that runs daily after price crawl chain completes.

   - For each ticker with open lots:
     a. Get the DailyPick linked to the BUY trade (via SimulatorTrade.daily_pick_id from the lot's trade_id)
     b. If no DailyPick or no stop_loss/take_profit_1, skip
     c. Get latest DailyPrice.close for that ticker
     d. Convert: `close_vnd = Decimal(str(float(daily_price_close))) * 1000` (nghìn đồng → VND)
     e. Check SL: if `close_vnd <= pick.stop_loss` → sell all remaining shares at close_vnd
     f. Check TP: if `close_vnd >= pick.take_profit_1` → sell all remaining shares at close_vnd
     g. Execute sell via create_trade() with source="ai_auto", user_notes="Auto-sell: SL hit" or "Auto-sell: TP hit"
   - Return list of executed sell results

   CRITICAL: The SL/TP values on DailyPick are in VND. DailyPrice.close is in nghìn đồng. MUST multiply close × 1000 before comparing.

   For finding which DailyPick is linked to a position:
   - Query SimulatorTrade where ticker_id matches, side="BUY", daily_pick_id IS NOT NULL, ordered by trade_date DESC, limit 1
   - Use that pick's stop_loss and take_profit_1
