# Plan 16-02: Frontend — WebSocket Hook + ConnectionStatus + PriceFlash

## Goal
Create the frontend WebSocket integration: auto-connecting hook, connection status indicator, and price flash animation on existing tables.

## Context
- Per D-16-05: React hook `useRealtimePrices(symbols)` with auto-connect/reconnect
- Per D-16-08: Flash green/red on price cells, connection status in navbar
- Per 16-UI-SPEC: ConnectionStatusIndicator states, PriceFlashCell animation specs
- Typography: font-semibold only, sizes 12/14/18/24px
- WebSocket URL: `ws://host/ws/prices` (derive from API_BASE)

## Tasks

### T-16-03: useRealtimePrices hook
Create `frontend/src/lib/use-realtime-prices.ts`:
- WebSocket URL derived from API_BASE (replace http→ws, append /ws/prices)
- State: `prices: Record<string, RealtimePrice>`, `status`, `subscribedCount`
- Connect on mount, send subscribe message when symbols change
- Auto-reconnect with exponential backoff: 1s→2s→4s→8s→16s→30s max
- Handle message types: `price_update`, `heartbeat`, `market_status`
- Disconnect on unmount (cleanup)
- Export `RealtimePrice` type and `UseRealtimePricesReturn` interface

### T-16-04: ConnectionStatusIndicator component
Per UI-SPEC Component 1:
- Create `frontend/src/components/connection-status.tsx`
- Props: `status: "connected" | "reconnecting" | "disconnected" | "market_closed"`, `subscribedCount: number`
- Renders: 8px colored circle + label text
  - Connected: green-500 circle + "Live" + animate-pulse
  - Reconnecting: yellow-500 circle + "Đang kết nối..."
  - Disconnected: red-500 circle + "Mất kết nối"
  - Market closed: muted circle + "Thị trường đóng"
- Tooltip: "Kết nối WebSocket: ... • X mã đang theo dõi"
- Mobile: icon-only below sm breakpoint (hide label)
- Add to navbar right side

### T-16-05: PriceFlashCell component
Per UI-SPEC Component 2:
- Create `frontend/src/components/price-flash-cell.tsx`
- Props: `value: number`, `previousValue?: number`, `children: ReactNode`
- On value change (useEffect comparing previous):
  - If increased: flash green background
  - If decreased: flash red background
  - Transition: `transition-colors duration-1000`
- Respects `prefers-reduced-motion` (skip animation)
- Wrapper: `rounded px-1 -mx-1`

### T-16-06: Integration into existing pages
- **WatchlistTable**: Wrap price column cells in PriceFlashCell, use useRealtimePrices with watchlist symbols to get live prices
- **HoldingsTable**: Wrap current price cells in PriceFlashCell, use useRealtimePrices with held symbols
- **Ticker detail page**: Use useRealtimePrices([symbol]) to show live price in header
- **Navbar**: Add ConnectionStatusIndicator with status from useRealtimePrices (need a global provider or shared state)

For global WebSocket state: Create a lightweight context provider `RealtimePriceProvider` that wraps the app layout. All pages share the same WebSocket connection. Individual components read from context.

TypeScript must compile with 0 errors.
