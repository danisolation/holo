| 2 | Kháng cự 1 | `resistance_1` | `text-[#ef5350]` with `bg-[#ef5350]/10` badge |
| 3 | Điểm xoay | `pivot_point` | `text-muted-foreground` with `bg-muted` badge |

### Fibonacci Column (right)

Display includes swing high/low anchors for context (Agent's Discretion decision: YES, show them — they explain where the Fib levels come from):

| Row | Label | Value Source | Color |
|-----|-------|-------------|-------|
| 1 | Đỉnh gần nhất | Derived: `max(fib_236, fib_382, fib_500, fib_618)` → swing_high via reverse calc, OR calculate from: `swing_high = fib_236 / 0.236 * (1 - 0.236) + fib_236` — simplified: show only if backend adds swing_high/swing_low, otherwise omit | `text-[#ef5350]` (no badge) |
| 2 | Fib 61.8% | `fib_618` | `text-foreground` |
| 3 | Fib 50% | `fib_500` | `text-foreground` |
| 4 | Fib 38.2% | `fib_382` | `text-foreground` |
| 5 | Fib 23.6% | `fib_236` | `text-foreground` |
| 6 | Đáy gần nhất | Derived from Fib levels (see note below) | `text-[#26a69a]` (no badge) |

**Swing High/Low derivation:** Since CONTEXT.md stores only fib_236/382/500/618 (not swing_high/swing_low directly), derive them:
- `swing_low = fib_236 - 0.236 * (fib_618 - fib_236) / (0.618 - 0.236)` → simplified: `swing_low = (fib_500 - 0.5 * fib_236 / 0.236)` — OR more cleanly:
- Given `level = swing_low + (swing_high - swing_low) * pct`:
  - From fib_500: `fib_500 = swing_low + range * 0.5` → `range = (fib_500 - swing_low) / 0.5`
  - From fib_236: `fib_236 = swing_low + range * 0.236`
  - Solving: `swing_low = (fib_500 * 0.236 - fib_236 * 0.5) / (0.236 - 0.5)` = `(fib_236 * 0.5 - fib_500 * 0.236) / (0.5 - 0.236)`
  - `swing_high = swing_low + (fib_500 - swing_low) / 0.5`
- **Implementation note:** If derivation adds complexity, simply omit swing high/low rows. The 4 Fibonacci levels alone are sufficient. This is an Agent's Discretion area — prefer showing them for user context.

## Data Contract (API ↔ Frontend)

### IndicatorData Type Extension

9 new nullable fields added to `IndicatorData` interface in `lib/api.ts`:

```typescript
export interface IndicatorData {
  // ... existing 18 fields ...

  // Phase 18: Support & Resistance Levels
  pivot_point: number | null;
  support_1: number | null;
  support_2: number | null;
  resistance_1: number | null;
  resistance_2: number | null;
  fib_236: number | null;
  fib_382: number | null;
  fib_500: number | null;
  fib_618: number | null;
}
```

Field names match database columns defined in CONTEXT.md.

### Data Extraction Pattern

The component extracts the **latest row** (most recent date) from the indicator data array for display:

```typescript
// Get latest S/R values from most recent date
const latest = indicatorData
  .filter(d => d.pivot_point != null)
  .sort((a, b) => b.date.localeCompare(a.date))[0];
```

If `latest` is undefined (no S/R data computed yet), render empty state.

### No Additional API Endpoint

S/R data is served via the existing `GET /api/analysis/{symbol}/indicators` endpoint. No new API endpoint required — the 9 new columns are included in the existing `IndicatorResponse` schema automatically after the backend migration.

## States

### Loading State

While `useIndicators` is loading, show a `Skeleton` placeholder matching the card dimensions:

```tsx
<Skeleton className="h-[220px] rounded-xl" />
```

Height 220px approximates the rendered card height (header + 5 pivot rows + column header + padding).

### Data Present State

Full card with Pivot + Fibonacci columns as described in Layout section.

### Empty State (no S/R data)

When indicator data exists but S/R fields are all null (e.g., new ticker with < 20 days of data for Fibonacci warm-up):

```tsx
<Card>
  <CardContent className="py-8 text-center">
    <p className="text-sm text-muted-foreground">
      Chưa đủ dữ liệu để tính hỗ trợ & kháng cự
    </p>
    <p className="text-xs text-muted-foreground/60 mt-1">
      Cần ít nhất 20 ngày giao dịch
    </p>
  </CardContent>
</Card>
```

### Partial Data State

When pivot data exists (needs only 1 prior day) but Fibonacci is null (needs 20 days):
- Render Pivot column normally
- Fibonacci column shows: `Cần ít nhất 20 ngày dữ liệu` in muted text, centered in the column area
