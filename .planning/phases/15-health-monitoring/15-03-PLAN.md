# Plan 15-03: Frontend — Gemini Usage Card + Pipeline Timeline

## Goal
Add two new components to the health dashboard: Gemini API usage visualization and pipeline execution timeline.

## Context
- Per 15-UI-SPEC: GeminiUsageCard with progress bars + breakdown + 7-day trend
- Per 15-UI-SPEC: PipelineTimeline with horizontal bar chart (Recharts) + date navigation
- Typography: font-semibold only (never font-bold), sizes 12/14/18/24px

- Existing health page has HealthStatusCards, DataFreshnessTable, DbPoolStatus, JobTriggerButtons, ErrorRateChart

## Tasks

### T-15-06: API types + fetch functions + hooks
Add to `api.ts`:
```typescript
// Gemini usage types
interface GeminiUsageTodayBreakdown { analysis_type: string; requests: number; tokens: number }
interface GeminiUsageToday { requests: number; tokens: number; limit_requests: number; limit_tokens: number; breakdown: GeminiUsageTodayBreakdown[] }
interface GeminiUsageDaily { date: string; tokens: number; requests: number }
interface GeminiUsageResponse { today: GeminiUsageToday; daily: GeminiUsageDaily[] }

// Pipeline timeline types
interface PipelineStep { job_id: string; job_name: string; started_at: string; duration_seconds: number | null; status: string }
interface PipelineRun { date: string; total_seconds: number; steps: PipelineStep[] }
interface PipelineTimelineResponse { runs: PipelineRun[] }
```

Add fetch functions: `fetchGeminiUsage(days=7)`, `fetchPipelineTimeline(days=7)`
Add hooks: `useGeminiUsage(days)`, `usePipelineTimeline(days)`
