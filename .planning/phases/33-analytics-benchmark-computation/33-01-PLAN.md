**Helper method `_get_run(self, run_id: int) -> BacktestRun`:**
Query BacktestRun by id. Raise HTTPException(404) if not found. Raise HTTPException(400, "Backtest not completed yet") if status != COMPLETED.

**Method 1: `async def get_performance_summary(self, run_id: int) -> dict`** (BENCH-02)
Compute from closed trades for this run_id:
- `total_trades`: COUNT of closed trades
- `wins`: COUNT where realized_pnl > 0
- `losses`: total - wins
- `win_rate`: wins / total * 100, round 2 decimals
- `total_pnl`: SUM(realized_pnl), round 2
- `total_pnl_pct`: total_pnl / initial_capital * 100, round 2 (get initial_capital from BacktestRun)
- `avg_pnl_per_trade`: total_pnl / total_trades, round 2

For max drawdown, query BacktestEquity for this run_id ordered by date ASC. Iterate equity curve:
- Track `peak` (highest total_equity seen so far)
- At each point, `drawdown = total_equity - peak`
- Track `max_drawdown_vnd` = minimum drawdown value (most negative)
- `max_drawdown_pct` = max_drawdown_vnd / peak_at_that_point * 100, round 2

For Sharpe ratio, collect `daily_return_pct` values from BacktestEquity (skip None values):
- `mean_return` = average of daily returns
- `std_return` = standard deviation of daily returns
- `sharpe_ratio` = (mean_return / std_return) * sqrt(252) if std_return > 0 else 0.0, round 4
- Use Python's `statistics.mean()` and `statistics.stdev()` (or `math.sqrt` + manual calc). Use `import math` for sqrt.

Return dict with all fields above.

**Method 2: `async def get_benchmark_comparison(self, run_id: int) -> dict`** (BENCH-01)
1. Call `_get_run(run_id)` to get start_date and end_date
2. Query BacktestEquity for this run_id ordered by date ASC → AI equity curve
3. Fetch VN-Index OHLCV via `self.crawler.fetch_ohlcv("VNINDEX", str(run.start_date), str(run.end_date))`
4. From VN-Index DataFrame: first row's `close` = `vnindex_base_price`. For each date, compute `vnindex_return_pct = (close / vnindex_base_price - 1) * 100`
5. Build a dict mapping date → vnindex data for O(1) lookup
6. Build aligned time-series: iterate AI equity dates. For each date, include:
   - `date`: ISO format string
   - `ai_equity`: float(total_equity)
   - `ai_return_pct`: float(cumulative_return_pct) or 0.0
   - `vnindex_return_pct`: matched from VN-Index dict (or None if no VN-Index data for that date)
7. Compute summary:
   - `initial_capital`: float(run.initial_capital)
   - `ai_total_return_pct`: last AI equity's cumulative_return_pct
   - `vnindex_total_return_pct`: last VN-Index return
   - `outperformance_pct`: ai_total - vnindex_total
8. Return dict with summary fields + `data` list.

**IMPORTANT for VN-Index fetch:** Wrap the vnstock call in try/except. If fetch fails (network error, VNINDEX symbol not available), return the response with `vnindex_return_pct: None` for all data points and `vnindex_total_return_pct: None`. Log the error with loguru. Do NOT fail the entire endpoint.

**Method 3: `async def get_sector_breakdown(self, run_id: int) -> list[dict]`** (BENCH-03)
Join BacktestTrade with Ticker on ticker_id. Filter by run_id and CLOSED_STATUSES. Group by `func.coalesce(Ticker.industry, "Unknown")`. For each group:
- `sector`: the industry name
- `total_trades`: COUNT
- `wins`: COUNT where realized_pnl > 0
- `win_rate`: wins / total * 100, round 2
- `total_pnl`: SUM(realized_pnl), round 2
- `avg_pnl`: AVG(realized_pnl), round 2
Order by total_trades DESC.

Follow the exact pattern from `PaperTradeAnalyticsService.get_sector_analysis()` but filter by `BacktestTrade.run_id == run_id`.

**Method 4: `async def get_confidence_breakdown(self, run_id: int) -> list[dict]`** (BENCH-04)
Use SQLAlchemy `case()` expression to bucket `BacktestTrade.confidence`:
- `confidence <= 3` → "LOW (1-3)"
- `confidence <= 6` → "MEDIUM (4-6)"
- else → "HIGH (7-10)"

Filter by run_id and CLOSED_STATUSES. Group by bucket. For each:
- `bracket`: bucket label
- `total_trades`: COUNT
- `wins`: COUNT where realized_pnl > 0
- `win_rate`: wins / total * 100, round 2
- `avg_pnl`: AVG(realized_pnl), round 2
- `avg_pnl_pct`: AVG(realized_pnl_pct), round 2

Follow the exact pattern from `PaperTradeAnalyticsService.get_confidence_analysis()`.

**Method 5: `async def get_timeframe_breakdown(self, run_id: int) -> list[dict]`** (BENCH-05)
Use SQLAlchemy expression for holding days: `BacktestTrade.closed_date - BacktestTrade.entry_date` (PostgreSQL date - date = integer days).

Bucket with `case()`:
- `holding_days <= 5` → "SHORT (1-5d)"
- `holding_days <= 15` → "MEDIUM (6-15d)"
- else → "LONG (16+d)"

Filter by run_id, CLOSED_STATUSES, AND `BacktestTrade.entry_date.isnot(None)` AND `BacktestTrade.closed_date.isnot(None)` (need both dates for holding_days calc).

For each bucket:
- `bucket`: bucket label
- `total_trades`: COUNT
- `wins`: COUNT where realized_pnl > 0
- `win_rate`: wins / total * 100, round 2
- `avg_holding_days`: AVG(holding_days expression), round 1
- `total_pnl`: SUM(realized_pnl), round 2
- `avg_pnl`: AVG(realized_pnl), round 2
