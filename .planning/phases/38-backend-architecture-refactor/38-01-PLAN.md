# Plan 38-01: Split AIAnalysisService and BacktestEngine

## Wave 1: AIAnalysisService Decomposition (BCK-04)

### Task 1: Extract ContextBuilder
**What**: Move all `_get_*_context()` methods (5 methods, ~300 LOC) into `services/analysis/context_builder.py`
**Files**:
- CREATE `backend/app/services/analysis/__init__.py`
- CREATE `backend/app/services/analysis/context_builder.py` — ContextBuilder class with session, and all 5 context methods
- MODIFY `backend/app/services/ai_analysis_service.py` — remove extracted methods, import and use ContextBuilder via composition
- MODIFY `backend/app/services/backtest_analysis_service.py` — override ContextBuilder methods (subclass BacktestContextBuilder)

**Key Logic**:
- ContextBuilder.__init__(session) stores the DB session
- Methods: _get_technical_context, _get_fundamental_context, _get_sentiment_context, _get_combined_context, _get_trading_signal_context
- BacktestContextBuilder(ContextBuilder) overrides _get_technical_context, _get_combined_context, _get_trading_signal_context with date-filtered versions, adds __init__(session, run_id, as_of_date)

### Task 2: Extract GeminiClient

**Files**:
