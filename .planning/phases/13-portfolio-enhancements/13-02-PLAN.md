1. **Add `recalculate_lots` method to `PortfolioService`:**
   ```python
   async def recalculate_lots(self, ticker_id: int) -> None:
       """Delete all lots for ticker and replay trades to rebuild FIFO state.
       Per PORT-11: ensures lot consistency after trade edit/delete."""
   ```
   Implementation:
   - DELETE all rows from `lots` WHERE `ticker_id = ticker_id` using `delete(Lot).where(Lot.ticker_id == ticker_id)`
   - Query all trades for this ticker: `SELECT * FROM trades WHERE ticker_id = :id ORDER BY trade_date ASC, id ASC`
   - For each BUY trade: create a new `Lot(trade_id=trade.id, ticker_id=ticker_id, buy_price=trade.price, quantity=trade.quantity, remaining_quantity=trade.quantity, buy_date=trade.trade_date)`, add to session
   - For each SELL trade: get open lots ordered FIFO (oldest first by buy_date, then id), consume `remaining_quantity` FIFO until sell quantity is fulfilled (same logic as `_consume_lots_fifo` but operating on freshly created lots in memory)
   - `await self.session.flush()` to persist changes

2. **Add `update_trade` method to `PortfolioService`:**
   ```python
   async def update_trade(self, trade_id: int, side: str, quantity: int, price: float, trade_date: date, fees: float = 0) -> dict:
       """Update an existing trade and recalculate FIFO lots. Per PORT-11."""
   ```
   Implementation:
   - Query trade by id: `SELECT * FROM trades WHERE id = :trade_id`
   - If not found → raise `ValueError(f"Trade {trade_id} not found")`
   - Update fields: `trade.side`, `trade.quantity`, `trade.price`, `trade.trade_date`, `trade.fees`
   - Call `await self.recalculate_lots(trade.ticker_id)`
   - `await self.session.commit()`
   - Query the ticker symbol for response: `SELECT symbol FROM tickers WHERE id = trade.ticker_id`
   - Return trade dict in same format as `record_trade`

3. **Add `delete_trade` method to `PortfolioService`:**
   ```python
   async def delete_trade(self, trade_id: int) -> dict:
       """Delete a trade and recalculate FIFO lots. Per PORT-11."""
   ```
   Implementation:
   - Query trade by id, raise ValueError if not found
   - Store `ticker_id` before deleting
   - `await self.session.delete(trade)`
   - `await self.session.flush()`
   - Call `await self.recalculate_lots(ticker_id)`
   - `await self.session.commit()`
   - Return `{"deleted": True, "trade_id": trade_id}`

"""CSV import service: parse VNDirect/SSI broker exports into trades.
Per PORT-12: auto-detect format, validate, dry-run preview, commit."""

1. **`CSVImportService` class:**
   - `__init__(self, session: AsyncSession)` — stores session for ticker validation
   - `detect_format(self, header: list[str]) -> str` — returns "VNDirect" | "SSI" | "unknown"
     - VNDirect indicators: any header containing "Mã CK" or "Mã chứng khoán", plus "Khối lượng" or "KL"
     - SSI indicators: header containing "Symbol" and "Quantity" (English headers)
   - `parse_rows(self, content: str) -> tuple[str, list[dict]]` — reads CSV string, detects format, maps columns to standard fields
     - VNDirect mapping: `Mã CK`→symbol, `Loại GD`→side (Mua→BUY, Bán→SELL), `Khối lượng`→quantity, `Giá`→price, `Ngày GD`→trade_date (parse DD/MM/YYYY), `Phí GD`→fees
     - SSI mapping: `Symbol`→symbol, `Side`→side, `Quantity`→quantity, `Price`→price, `Trade Date`→trade_date (parse YYYY-MM-DD), `Fees`→fees
     - Strip whitespace, handle quoted values (Python csv module handles this)
     - Return tuple of (format_detected, list_of_row_dicts)
   - `async validate_rows(self, rows: list[dict]) -> list[CSVPreviewRow]` — validates each row
     - Check symbol exists in tickers table (batch query all unique symbols first)
     - Check quantity > 0, price > 0
     - Check trade_date is not in the future
     - Check side is BUY or SELL
     - Detect duplicate rows (same date+symbol+side+quantity) → warning
     - Return list of `CSVPreviewRow` with status "valid"/"warning"/"error" and message
   - `async dry_run(self, content: str) -> CSVDryRunResponse` — parse + validate, return preview
   - `async import_trades(self, content: str) -> dict` — parse, validate, record valid trades
     - Call `PortfolioService.record_trade()` for each valid row
     - Track unique ticker_ids for recalculation count
     - Return `{"trades_imported": N, "tickers_recalculated": M}`
