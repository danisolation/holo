**GREEN phase — Implement:**

1. **Add `get_performance_data` method to `PortfolioService`:**
   ```python
   async def get_performance_data(self, period: str = "3M") -> list[dict]:
       """Compute daily portfolio value snapshots for performance chart.
       Per PORT-09: join holdings × DailyPrice for each date."""
   ```
   Implementation approach:
   - Compute `start_date` from period: `1M`→30d, `3M`→90d, `6M`→180d, `1Y`→365d, `ALL`→earliest trade date
   - Query all trades ordered by `trade_date ASC`
   - Get unique `ticker_ids` from trades
   - Bulk-fetch daily prices: `SELECT ticker_id, date, close FROM daily_prices WHERE ticker_id IN (...) AND date BETWEEN start AND end ORDER BY date`
   - Build price lookup: `Dict[(ticker_id, date), close_price]`
   - Get unique sorted dates from the price data
   - Iterate through dates chronologically, maintaining a `positions: Dict[ticker_id, int]` dict
   - For each date, apply any trades on that date (BUY adds, SELL subtracts), then compute `value = Σ(positions[tid] × prices[(tid, date)])` for all tickers with position > 0
   - Skip dates where no position exists (before first trade)
   - Return `[{"date": d.isoformat(), "value": round(float(value), 2)}]`
   - Use `from datetime import timedelta` for date arithmetic

2. **Add `get_allocation_data` method to `PortfolioService`:**
   ```python
   async def get_allocation_data(self, mode: str = "ticker") -> list[dict]:
       """Compute portfolio allocation by ticker or sector.
       Per PORT-10: group holdings by ticker or sector."""
   ```
   Implementation:
   - Call `self.get_holdings()` to get current holdings
   - Filter holdings where `market_value is not None`
   - Compute `total_value = Σ(market_value)`
   - If `mode == "ticker"`: each holding becomes an item `{name: symbol, value: market_value, percentage: round(market_value / total_value * 100, 2)}`
   - If `mode == "sector"`: group holdings by `sector` field (use "Khác" for None sector), sum market_values per sector, compute percentage
   - Sort by value descending
   - Return list of dicts
